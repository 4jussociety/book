# 물리치료/재활 일정 관리 시스템 화면설계서 (Text Wireframe)
# 이 문서는 '기능명세서'와 '페이지 구성 가이드'를 기반으로 각 화면의 UI 구성요소, 배치, 인터랙션을 정의합니다.

## 1. 공통 레이아웃 (Global Layout)

### 1.1 네비게이션 바 (GNB) - 좌측 또는 상단
- **로고**: 시스템 타이틀 (클릭 시 캘린더 이동).
- **메뉴**:
  1. **예약 관리** (기본/메인)
  2. **환자 관리**
  3. **통계**
- **우측 영역**:
  - **로그인 사용자 정보**: `[홍길동 치료사] 프로필 아이콘`
  - **로그아웃 버튼**

---

## 2. 로그인 페이지 (Login)

### 2.1 화면 구성
- **중앙 배치 컨테이너**:
  - **로고/서비스명**
  - **ID 입력 필드**: Placeholder="아이디를 입력하세요"
  - **PW 입력 필드**: Placeholder="비밀번호를 입력하세요" (Type=password)
  - **[로그인] 버튼**: (Full width, Primary Color)
  - **에러 메시지 영역**: 로그인 실패 시 노출 (예: "아이디 또는 비밀번호가 일치하지 않습니다.").

### 2.2 인터랙션
- 엔터키 입력 시 로그인 시도.
- 5회 실패 시 "1분간 계정 잠금" 메시지 표시 및 버튼 비활성화.

---

## 3. 예약 관리 페이지 (메인 캘린더)

### 3.1 상단 헤더 (Toolbar)
- **좌측**:
  - `[<]` `[오늘]` `[>]` 날짜 네비게이션.
  - **현재 날짜 표시**: "2026년 2월 15일 ~ 2월 21일 (8주차)".
- **중앙/우측**:
  - **치료사 필터**: 체크박스 또는 토글 버튼 나열 (예: `[v] 김치료(파랑)`, `[v] 박재활(초록)`).
  - **검색창**: "캘린더 내 찾기" (Placeholder="예약자명, 번호, 메모..."). icon=`돋보기`.
  - **[잠금 모드] 토글**: On/Off 시 드래그 동작이 '잠금(Block)' 생성으로 변경.

### 3.2 메인 캘린더 영역 (Calendar Body)
- **좌측 축 (Time Axis)**: 08:00 ~ 21:00 (1시간 단위 라벨, 10분 단위 그리드 선).
- **상단 축 (Day/Therapist Axis)**:
  - 날짜 헤더: "월(2/16)", "화(2/17)" ...
  - 치료사 서브 헤더: 날짜 아래 치료사별 컬럼 분할 (예: 김치료 | 박재활).
- **그리드 셀 (Cell)**:
  - 10분 단위 높이.
  - Hover 시: `+` 아이콘을 흐릿하게 표시 (클릭 유도).
  - 마우스 클릭/드래그 시: 파란색 점선 박스(Ghost Event)로 선택 영역 표시.

### 3.3 예약 박스 (Event Card)
- **스타일**:
  - 배경색: 치료사 지정 컬러 (Light tone).
  - 좌측 스트립: 상태 컬러 (파랑-예정, 초록-완료, 회색-취소, 빨강-노쇼).
- **내용**:
  - **1줄**: 이름 `강하늘` + `#12`(회색 뱃지) + `[5회차]`.
  - **2줄**: 메모 내용 (말줄임...).
- **상호작용**:
  - **클릭**: 상세/수정 모달 오픈.
  - **드래그**: 시간/치료사 이동.
  - **리사이즈(하단 핸들)**: 시간 늘리기/줄이기.

---

## 4. 예약 생성/수정 모달 (Modal Flow)

### 4.1 모달 A: 예약 타입 선택
- **타이틀**: "새 예약 생성"
- **내용**:
  - `[초진 (신규 환자)]` 버튼 (큰 사이즈).
  - `[재진 (기존 환자)]` 버튼 (큰 사이즈).
- **하단**: `[취소]` 버튼.

### 4.2 모달 B: 환자 검색 (재진 선택 시)
- **타이틀**: "환자 검색"
- **검색창**: Auto-focus. (이름/번호/연락처 입력).
- **결과 리스트**:
  - 선택 시 해당 환자로 **모달 C** 진입.
  - 키보드(위/아래/엔터) 조작 지원 권장.

### 4.3 모달 C: 상세 입력 (Input Form)
*초진/재진 공통 폼 (재진은 환자정보 readonly)*

- **헤더**:
  - 타이틀: "예약 상세"
  - 상태값(우측 상단): Dropdown (`예정/완료/취소/노쇼`).
- **환자 정보 섹션**:
  - **이름**: (초진: 입력 / 재진: 텍스트).
  - **고객번호**: (초진: 자동생성 Placeholder / 재진: `#12`).
  - **연락처**: (초진: 필수 입력 / 재진: 텍스트).
  - **성별/생년월일**: 라디오버튼/Datepicker.
- **예약 정보 섹션**:
  - **일시**: Date / Start Time ~ End Time (10분 단위 Select).
  - **치료사**: Dropdown (현재 트랙 자동 선택).
  - **내용(Note)**: Textarea (여러 줄).
    - *Tip*: "이전 기록 불러오기" 버튼 또는 재진 시 자동 채움.
- **하단 액션 바**:
  - `[문자 생성]` (아이콘 버튼).
  - `[삭제]` (기존 예약 수정 시만 노출, 빨간색).
  - `[취소]`, `[저장]` (Primary).

### 4.4 기록 히스토리 (재진 시 모달 C 하단)
- **타이틀**: "과거 진료 기록"
- **리스트**:
  - `2025-01-20 (김치료)`: "허리 통증 호소..."
  - `2025-01-10 (박재활)`: "초진. 엑스레이 촬영 필요..."

---

## 5. 환자 관리 페이지

### 5.1 상단 검색/필터 바
- **검색창**: "이름, 번호, 연락처 검색".
- **필터**: `[전체 치료사]` Dropdown.
- **우측 버튼**: `[+ 신규 환자 등록]` (클릭 시 캘린더 초진 모달과 동일한 폼 팝업).

### 5.2 환자 목록 테이블 (Data Table)
- **컬럼**:
  1. **고객번호** (#12)
  2. **이름** (클릭 시 상세 모달)
  3. **성별/생년월일** (남/90.01.01)
  4. **연락처**
  5. **최초 방문일**
  6. **최근 방문일**
  7. **담당 치료사** (최근 예약 기준)
  8. **관리**: `[예약 잡기]` 버튼.
- **Paging**: 하단 1 2 3 4 5 ...

### 5.3 인터랙션
- **예약 잡기 클릭**: 캘린더 페이지로 이동 -> 예약 모달(재진)이 바로 뜨거나, 드래그 대기 상태로 전환.

---

## 6. 통계 페이지

### 6.1 상단 컨트롤
- **기간 선택**: `[오늘]` `[이번 주]` `[이번 달]` `[날짜 지정]`.
- **내보내기**: `[엑셀 다운로드]` 버튼.

### 6.2 대시보드 (Summary Cards)
- 4개의 큰 카드 배치:
  1. **총 예약**: 00건
  2. **완료**: 00건
  3. **노쇼/취소**: 00건 (노쇼율 0%)
  4. **신규 환자**: 0명

### 6.3 차트/테이블 영역
- **시간대별 분포**: 막대 차트 (가로축: 09시~20시 / 세로축: 예약 수).
- **치료사별 실적 테이블**:
  - 컬럼: 치료사명 | 총 예약 | 완료 | 매출액(추정) | 재방문율.

---

## 7. 모바일/반응형 고려사항 (권장)
- **캘린더**: 모바일에서는 '일간(Day) 뷰' 또는 '리스트(Schedule) 뷰'로 전환 고려.
- **모달**: 모바일에서는 화면 전체를 덮는 Full-screen Bottom Sheet 형태 권장.
